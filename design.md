#### 设计文档

实现的功能，以及未实现的功能

- [x] 圆，正方形，三角形等简单图形绘制 （√）
- [x] 房子，太阳等简单图形组合图形绘制  （√）
- [x] 设置图形颜色（√）
- [x] 填充图形（√）
- [x] 设置图形粗细（√）
- [x] 删除指定图形（√）
- [x] 清空画布（√）
- [ ] 橡皮擦
- [ ] 自由绘画

1. 橡皮擦功能和图形删除功能不同，普通的删除功能直接操作图形对象，直接删除这个对象就能实现，但是橡皮擦功能操作的是像素级位图，在目前的系统设计框架中还没有想好去如何使用语音随意操作像素级位图。
2. 自由绘画设计绘制不规则曲线，并且依赖连续的手部轨迹，但是目前系统的没有想好如何实现通过语音实现不规则连续曲线轨迹绘制。
